# 류웅선의 포트폴리오(겸 이력서)

## 소개

취업과 관련되어 지금까지 진행한 사항에 대해서 소개합니다.<br/>
개인적인 특징과 인적사항을 포함합니다. 깃허브 주소와 블로그의 주소, 그리고 연락가능한 이메일 주소를 포함하고 있습니다.

### 방향성

가능하다면 꾸준히 업데이트를 하는 것을 목표로 합니다.<br/>
프로젝트가 새로 진행될 때마다 주기적으로 업데이트를 하겠습니다.
혹은 제가 시도하고 싶은 기술이 생긴다면요!
